hexo文章设置标题、分类和标签

发表于 2022-04-28 分类于 技术 
#hexo n "文档1"   这样可以在服务器端生成一个带有title的.md文件，编辑后hexo g就可以直接发布。
如果是在PC上编写，需要自己在md文件前部加上title内容。并且上传后需要hexo g/hexo d才能发布。

在md文档的头部添加title/date/categories/tags标注
---
title: 标题
date:
categories: 
- 技术
- 笔记    二级分类
tags: 
- hexo
- markdown
---

使用picgo+阿里云OSS自建markdown图床

发表于 2022-04-28 分类于 技术 ， 笔记

  1. 登录阿里云，实名认证，启用access key

  2. 开通阿里云对象存储服务，创建bucket（使用广州3节点）并获得endpoint: oss-cn-guangzhou

  3. 暂不购买资源包（40G，每年9元），使用按流量计费模式（每月10G 1.2元）

  4. Mac上下载安装picgo应用，配置图床，输入ak/sk/bucket/endpoint，确定后提示配置成功。
